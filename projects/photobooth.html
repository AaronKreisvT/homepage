Photobooth

Privacy-friendly DSLR photobooth with tokenized QR links and a kiosk UI, built on Raspberry Pi.

Overview

  • Stack: Python, Tkinter, Pillow, qrcode, subprocess, OpenSSH, Bash, gphoto2, Raspberry Pi, HTML, CSS
  • Status: Work in Progress.
  • Repo: GitHub

Notes

As a broke physics student with a love for DIY and tinkering, I couldn't justify spending hundreds of euros on a commercial photo booth for parties and events. So, I built my own — using the tools and knowledge I already had.

This project is a self-made photo booth, built around a wooden frame, a Raspberry Pi with a touchscreen, and a Nikon DSLR camera connected via USB. The result is a compact, reliable, and fun little system that lets guests take high-quality photos with a single tap — and instantly get their picture via a personal QR code.

The software is kept intentionally lightweight and modular:

  • The client side (running on the Raspberry Pi) handles image capture, display, and upload.
  • The server side hosts a simple website where guests can retrieve their photos by scanning the QR code.

No accounts, no cloud services, no ads — just local hardware, open protocols, and full control.
The booth has already been used at birthdays, graduation parties, and student events — and it's always a hit.
