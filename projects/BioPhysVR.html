Virtual Biophysics Microscopy Lab

An interactive VR laboratory for understanding optical microscopy in biophysics.

Overview

  • Stack: VR
  • Status: Planing
  • Repo: GitHub

Notes

This project recreates a fully modular microscope environment in virtual reality, where users can assemble optical components such as lenses, apertures, objectives, and filter cubes to construct brightfield, darkfield, and fluorescence microscopes. Standard biological samples (e.g., red blood cells) can be observed under varying conditions, including different buffer solutions and fluorescent labeling.
Instead of performing computationally expensive real-time wave-optics simulations, the system uses curated image datasets combined with controlled stochastic variations (focus, noise, drift) to realistically emulate microscopy outcomes.
The goal is to bridge optical theory and laboratory practice, offering both an educational tool and a platform for studying conceptual understanding of imaging physics.
