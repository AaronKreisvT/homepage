Homepage

Minimal personal site to practise HTML and showcase my work.

Overview

  • Stack: HTML, CSS (custom), JavaScript (vanilla)
  • Hosting: Self-managed Linux server (Strato domain, DynDNS via ddclient)
  • Status: Finished; iteratively refining styles and content
  • Repo: GitHub

Notes

The site is intentionally simple and static. Shared header/footer are included via a small JS helper, links are root-relative to work from subpages, and a theme toggle provides light/dark mode with a smooth transition. The CV page embeds a PDF viewer. Deployment is done from a local repo to the webroot via rsync. Some project pages may be placeholders while I collect ideas.
